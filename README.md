# Portfólio Pessoal - Full Stack

Aplicação web desenvolvida para apresentar projetos, habilidades e trajetória como desenvolvedora, utilizando tecnologias modernas de front-end e back-end.

---

## Objetivo

Construir uma aplicação completa (full stack) que sirva como vitrine profissional, demonstrando:

* Desenvolvimento de interfaces modernas e responsivas
* Integração com APIs
* Organização de código e boas práticas
* Conhecimento em desenvolvimento full stack

---

## Tecnologias Utilizadas

### Front-End

* React
* TypeScript
* HTML5
* CSS3

### Back-End

* Python (Flask)

### Banco de Dados

* SQLite / PostgreSQL

### Outros

* Git e GitHub
* Vite
* Consumo de APIs REST

---

## Funcionalidades

* Página inicial com apresentação pessoal
* Listagem de projetos com descrição e tecnologias
* Detalhamento de projetos
* Formulário de contato
* Integração com API para envio de mensagens
* Interface responsiva

## Como Executar o Projeto

### 1. Clonar o repositório

```bash id="k4b7mz"
git clone <url-do-repositorio>
```

---

### 2. Back-End

```bash id="p9c2wd"
cd backend
pip install -r requirements.txt
python app.py
```

---

### 3. Front-End

```bash id="u2n8qy"
cd frontend
npm install
npm run dev
```
